# Кондитерская лавка — Интернет-магазин на Django

Современное веб-приложение для продажи кондитерских изделий (торты, пирожные, печенье и др.). Проект реализован на фреймворке Django и предоставляет полный цикл обслуживания клиента: от просмотра каталога до оформления заказа, а также удобные инструменты управления для администратора.

## Ключевые возможности

### Функционал для покупателей
* **Каталог и навигация:** Просмотр товаров по категориям.
* **Топ продаж:** Автоматический вывод 5 самых популярных товаров на главной странице.
* **Акции и скидки:** Отдельный раздел с товарами, на которые установлена скидка.
* **Система корзины:** Добавление товаров, изменение их количества и расчет итоговой стоимости в реальном времени.
* **Личный кабинет:** Регистрация и авторизация пользователей, возможность редактирования профиля (ФИО, телефон, адрес).
* **Оформление заказа:** Простая форма оформления заказа с сохранением истории в базе данных.

### Функционал для администратора
* **Управление контентом:** Удобная работа с категориями и товарами через встроенную админ-панель Django.
* **Импорт из Excel:** Мощный инструмент для массовой загрузки категорий и товаров из `.xlsx` файлов, что значительно ускоряет наполнение магазина.
* **Учет заказов:** Просмотр поступивших заказов с детализацией по товарам и контактным данным клиентов.

## Технологический стек

* **Backend:** Python 3.x, Django 5.x
* **Database:** PostgreSQL
* **Frontend:** HTML5, CSS 
* **Библиотеки:** * `openpyxl` (обработка Excel-файлов)
    * `Pillow` (работа с изображениями)
    * `django-crispy-forms` (стилизация форм)
    * `python-decouple` (управление секретами)

## Структура проекта

* `catalog/` — основное приложение: товары, категории, поиск и логика импорта.
* `accounts/` — управление пользователями, кастомная модель пользователя и профили.
* `cart/` — логика корзины (хранение данных в сессиях).
* `orders/` — обработка и хранение заказов.
* `confectionery_shop/` — системные настройки проекта.
